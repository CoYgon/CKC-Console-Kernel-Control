# Güvenlik Politikası

## Desteklenen Sürümler

| Sürüm | Destekleniyor |
|-------|---------------|
| 3.x   | ✅ Evet        |
| < 3.0 | ❌ Hayır       |

## Güvenlik Açığı Bildirimi

CKC, kernel seviyesinde Windows API'lerine erişen bir araçtır. Yanlış kullanımı sistem kararlılığını etkileyebilir. Bir güvenlik açığı keşfederseniz:

1. **Public issue açmayın** - Güvenlik açıklarını doğrudan GitHub Issues'da bildirmeyin
2. CoYgon'a doğrudan iletişim kanallarından ulaşın
3. Mümkünse açığın detaylarını, etkilediği sürümleri ve varsa çözüm önerisini belirtin

## Önemli Uyarılar

- Bu araç yalnızca **eğitim ve sistem yönetimi** amaçlıdır
- Yetkisiz sistemlere karşı kullanımı yasa dışı olabilir
- Yönetici yetkisi gerektiren komutları dikkatli kullanın
- Proses dondurma/sonlandırma, servis durdurma gibi işlemler sistem kararlılığını etkileyebilir
- **Sorumluluk reddi**: Bu aracın kullanımından doğabilecek her türlü hasardan geliştirici sorumlu değildir
